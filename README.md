# 每日新闻摘要 · Daily News Digest

巡视全球主流报纸，覆盖**政治·国际 / 经济·财经 / 科技 / 社会·民生 / 灾害·突发**五大类，筛选去重后生成中文原创摘要并附原文链接。每期另附 **📚 概念观察** 栏目，从新闻中抽取宏观经济学 & 社会学核心概念配现实案例。
